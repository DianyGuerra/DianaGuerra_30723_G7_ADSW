## Resumen del SRS – Sistema de Cotizaciones *FastKote*

**FastKote** es un sistema web desarrollado para el negocio de catering *“Chichi Está de Fiesta”*, cuyo propósito es **automatizar y optimizar la generación de cotizaciones de eventos**.

El sistema permite gestionar clientes, paquetes, servicios, insumos y promociones, centralizando toda la información del negocio en una sola plataforma.

## Objetivo
Optimizar el proceso de cotización, reducir errores manuales y mejorar la eficiencia operativa del negocio.

### Funcionalidades principales
- Autenticación y control de acceso por roles  
- Gestión completa de cotizaciones  
- Envío de cotizaciones en PDF vía WhatsApp  

### Arquitectura
El sistema sigue una arquitectura **cliente-servidor**, con base de datos relacional y acceso mediante navegador web.
